LDP.js

LDP.js is an LDP server built using Node.js and MongoDB. It accepts most any RDF, but does have a few constraints:

  • Turtle is the only media type currently supported
  • LDP.js does not yet support non-RDF source.
  • LDP.js does not support ldp:DirectContainer or ldp:IndirectContainer.
  • Containment triples cannot be modified using PUT as recommended by the LDP 1.0 Candidate Recommendation.
  • The interaction model of a resource cannot be changed by modifying its rdf:type after creation. If the resource was created as a container, it will always be a container.
